# Task Blocking Feature Stories

This folder contains user stories and acceptance criteria for the **Task Blocking** feature. The primary story is `story-01-blocking.md`, which outlines the core functionality and edge cases.

## How to Use
- Review `story-01-blocking.md` for detailed acceptance criteria.
- Implement the feature following the technical documentation in `docs/technical/database-schema.md` and the product PRD.
- Write integration tests to cover all scenarios described in the story.
